Jawshan Kabeer (جوشن الكبیر‎) is a prayer that contains 1000 names and attributes of God. Jawshan means breastplate, chain mail or a similar type of armor. The prayer is transmitted from the Prophet Muhammad (PBUH) by Ahl al-Bayt .

Easy Dua is a progressive web application. You can read on web page and install on your mobile device.

After installing without internet connection it can run offline.

Features

  • Easy to read
  • Easy to use
  • Scrolls top-to-bottom (Not left-to-right)
  • Runs on web and mobile (Responsive)
  • Installable
  • Runs offline
  • Lightweight (About 200KB)
  • Lightning fast
  • Go to paragraph
  • Font family selection
  • Font size selection
  • Color selection
  • Background color selection
  • Multi language
  • Printable (Press Ctrl+P)

Icons

  • q Go to paragraph
  • i Program info popup
  • t Go to top
  • d Go to bottom
  • b Delete Bookmark
  • 33 Paragraph bookmark (Click to go to the paragraph)
  • s Settings (Select font, size, color, background, language)
  • 33 Paragraph number (Click to bookmark)
