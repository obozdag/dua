a Easy Quran

Easy Quran is a progressive web application. You can read on web page and install on your mobile device.

After installing without internet connection it can run offline.

Features

  • Easy to read
  • Easy to use
  • Scrolls top-to-bottom not left-to-right
  • Verse number before verse (Not after)
  • Runs on web and mobile (Responsive)
  • Installable
  • Runs offline
  • Lightweight (About 2.7MB)
  • Lightning fast
  • Sura selection
  • Juz selection
  • Page selection
  • Font family selection
  • Font size selection
  • Color selection
  • Background color selection
  • Multi language
  • Printable (Press Ctrl+P)

Icons

  • q Quran navigation (Goto surah, juz, page)
  • i Program info popup
  • t Go to top
  • d Go to bottom
  • b Delete Bookmark
  • s Settings (Select font, size, color, background, language)
  • p439 Page number (Click to bookmark)
  • i Page info

While starting and changing settings it may take a bit time. Please be patient. Than scrolling is lightning fast!

For feature requests or any bug reports please send email to info at fklavye dot net
